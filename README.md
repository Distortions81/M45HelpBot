Simple discord bot that automatically replies to some common keywords in questions.

Usage of M45HelpBot:

  -guildid string
  
        discord guild id
  
  -helpFilePath string
  
        Specify path to helps file. (default "helps.hlp")
  
  -staffid string

        discord role ID for moderator/staff

  -testmode
 
        skip throttle check

  -token string
  
        discord token
